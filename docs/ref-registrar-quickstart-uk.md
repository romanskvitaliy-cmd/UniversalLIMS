# Направлення REF — коротка інструкція реєстратору

> Пілот ЦКПХ (Житомир). Оновлено: 2026-05-31.

## Терміни

- **Справa** — один візит клієнта (`Order`). **Номер справи** (REF-…) присвоюється автоматично.
- **Проба** — один зразок у справі.
- **Бланк направлення REF-*** — PDF направлення на дослідження (окремий документ поруч із протоколом).

Детальніше: `docs/glossary-registration-uk.md`.

## Щоденний flow (1 проба + протокол)

### 1. Прийом справи (`/Orders/Create`)

1. Заповнити **замовника**.
2. У рядку проби обрати:
   - **Бланк направлення REF-*** → `REF-MOZ-001`
   - **Бланк протоколу** → напр. f327 / Food
3. Зберегти.

На кожну пробу створюються **2 документи**: направлення + протокол.

### 2. Мапінг полів (якщо ≥2 бланки)

На `MapOrderFields` об’єднайте спільні поля **один раз**:

| Що вводите | Приклад зв’язку |
|------------|-----------------|
| Дата відбору | `REF_SamplingDate` + `f327_SamplingDate` |
| Місце відбору | `REF_SamplingLocation` + `f327_SamplingPoint` |
| Найменування зразка | `REF_SampleName` + `Food_SampleName` |

Auto-merge працює лише за **точним збігом тегу** — решту об’єднуйте вручну.

### 3. Заповнення PDF (Fill)

1. Відкрити справу → **Заповнити** на REF-документі.
2. Поля з SSOT (`Customer.*`, `Order.ReferralNumber`, `Sample.Number`) підтягуються автоматично.
3. Решту — ввести або перевірити; **«Зберегти значення»**.
4. Те саме для протоколу (реєстраційні поля).

### 4. Друк і лабораторія

- **«Друк направлення»** на Details — PDF лише REF (для клієнта / архіву).
- **«Друк протоколів реєстрації»** — протоколи без REF.
- **«Відправити в лабораторію»** — у lab йде **протокол**; направлення лишається в реєстрації.

### 5. Паперове направлення від клієнта

1. Прийняти папір / скан (скан — backlog `OrderAttachment`).
2. **Перенести дані** у цифровий REF-MOZ-001 через Fill.
3. Далі — звичайний flow з протоколом.

## Якщо REF не видно в Create

Адмін має опублікувати шаблон — див. `docs/ref-moz-001-admin-setup.md`.

## Допомога

- Номер справи в UI = `Order.ReferralNumber` (не плутати з паперовим бланком на пробу).
- На пілоті: **1 REF на пробу** (не на всю справу).
